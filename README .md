python3 -m venv venv
source venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt


Explicación de cada comando:

1. python3 -m venv venv - Crea el entorno virtual (solo se hace una vez)
2. source venv/bin/activate - Activa el entorno virtual
3. pip install --upgrade pip - Actualiza pip a la última versión
4. pip install -r requirements.txt - Instala todas las dependencias del proyecto
